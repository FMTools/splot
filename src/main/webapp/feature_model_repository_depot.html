Repository of Real Feature Models

The models below were either extracted from academic publications and other relevant sources (on a best effort basis) or edited by their authors using SPLOT's online feature model editor. If you're are an author and wish to update your feature model's entry please use the editor tool. If you disagree with a repository entry please contact us. The main purpose of the repository is to encourage knowledge sharing among researchers and ultimately improve research quality.


 

The quality of the feature model repository is enforced by the following rules:

  • Consistency: All models are guaranteed to be consistent (contain at least one valid configuration)
  • Correctness: None of the models contain dead features
  • Transparency: All models identify their authors (or related literature) and provide some contact information
